# UX Journey MindStudy

Dokumen ini menjelaskan pengalaman pengguna saat menggunakan aplikasi MindStudy.

## Tujuan
Membantu pengguna belajar lebih fokus dan terstruktur.

## Alur Pengalaman
Pengguna memulai dari login, mengatur jadwal belajar, menggunakan fitur fokus, dan memantau progres belajar secara berkala.
